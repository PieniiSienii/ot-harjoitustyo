# Requirements

## Purpose of the application
The purpose of this application is to collect simple, anonymous customer feedback and provide basic analysis, such as average ratings and mood distribution for the visit.

## Users
The application has two types of users: customers, who submit feedback, and the organization, which views and analyzes the feedback.

## Core functionality of the basic version:
- **Customers**
  - Submit anonymous feedback ✓
  - Rate their experience (happy, neutral, sad) ✓
  - Answer 1-3 simple questions ✓
- **The organization**
  - View aggregated feedback ✓
  - See average ratings ✓
  - See distribution of mood ✓
  - See distribution of yes/no or multiple-choice answers
  - Filter feedback

## Possible future improvements
- Allow customers to create a user account to view their own past feedback
- Display simple trend analysis, e.g., average ratings over time
- Send notifications or warnings if many negative feedbacks are received consecutively
- Add additional questions dynamically configurable by the organization
- Generate simple charts or graphs for visual analysis of feedback
